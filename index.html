Pre-Stimulus Event Related Potential(ERP) Effect

• Collected data from online datasets (sample datasets from mne and from the VEPESS dataset) and participated in collecting electroencephalogram data of several UCSD students

• Cleaned data using PCA and Autoreject and Processed data:
-- Applied general alpha band width filter to the data, from 8Hz to 12Hz
-- Used Hilbert transform to transfer the data from time domain to phase domain
-- Did the epoch around each event, which is select 1s of data around each event

• Evaluated the relationship between pre-stimulus phase (phase before the event happens, usually about 0.1ms) and ERP. This relationship belongs to circular-linear statistics.
